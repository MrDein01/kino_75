# Публикация сайта на обычный PHP-хостинг

Этот вариант рассчитан на хостинг с PHP 8.1+ и записью в папки проекта: Beget, Timeweb, Reg.ru, Sprinthost, обычный VPS и т.п.

## Куда загружать

Загрузите содержимое папки `kultura75_site` в корневую папку сайта, например:

- `public_html/`
- `www/`
- `htdocs/`

Важно: внутри корня должны лежать сразу `index.php`, `admin/`, `assets/`, `includes/`, `data/`, `uploads/`.

## Права на папки

Для работы админки нужны права записи:

- `data/`
- `data/news.json`
- `uploads/`

Обычно достаточно прав `755` для папок и `644` для файлов. Если админка не сохраняет новости, поставьте для `data` и `uploads` права `775` или временно `777` и затем уточните настройки у хостинга.

## Админка

Адрес:

`/admin/`

Логин:

`admin`

Пароль:

`Kultura75!`

После публикации лучше сменить пароль в файле `includes/config.php`.

## Vercel

На Vercel эта PHP-админка не сможет сохранять новости в `data/news.json`, потому что файловая система там read-only. Для Vercel нужна база данных/облачное хранилище. Этот архив предназначен именно для обычного PHP-хостинга.
